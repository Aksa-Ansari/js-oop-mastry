## ‚≠ê What is Prototype?
Prototype is a place where JavaScript stores **common methods**.
When an object does not find a method in itself, JavaScript looks into its prototype.

---

## ‚≠ê Why Prototype is Used?
- To share methods between objects  
- To save memory  
- To avoid duplicate code  

---

## ‚≠ê Prototypal Inheritance
Prototypal inheritance means one object can **access properties or methods of another object**.

JavaScript uses prototype chain to do this.

---
